ACARA 2:

BIN AVERAGE DATA CTD MENGGUNAKAN SBE DATA PROCESSING

Acara 2 ini berjudul "Bin Average Data CTD Menggunakan SBE Data Processing" dan merupakan bagian dari Praktikum Pengolahan Data Perikanan dan Kelautan. Modul ini mencakup teknik bin averaging untuk data CTD (Conductivity, Temperature, Depth), yaitu teknik pemrosesan data yang mengelompokkan data berdasarkan kedalaman atau jarak waktu, lalu menghitung rata-rata untuk setiap kelompok tersebut. Tujuan dari Modul ini adalah agar mahasiswa mampu melakukan Quality Control (QC) data CTD dan memvisualisasikan data yang diperoleh.
